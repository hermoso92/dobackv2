# Assets Directory

Este directorio contiene recursos estáticos utilizados en la aplicación DobackSoft V2, tales como:

- Imágenes y logos
- Iconos
- Fuentes
- Otros recursos estáticos

## Uso

Para utilizar un recurso de este directorio:

```javascript
import miRecurso from '../assets/miRecurso.png';
```

## Convenciones de Nomenclatura

- Utilizar kebab-case para nombres de archivos (ej: `logo-principal.png`)
- Prefijos recomendados:
  - `icon-` para iconos
  - `logo-` para logos
  - `bg-` para fondos (backgrounds)
  - `img-` para imágenes generales
